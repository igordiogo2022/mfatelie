# MF Ateliê🎀

### 📄Descrição
Site para uma vendedora de laços, tiaras e outros acessórios.

<hr>

### 🎮Funcionalidades
- Catálogo de produtos
- Interface atrativa
- Filtrar produtos
- Página para contato

<hr>

### 🎯Objetivo
Projeto desenvolvido para melhor divulgação e exibição de produtos, permitido uma melhor visualização para o cliente e facilitação para o processo de atendimento ao cliente.
